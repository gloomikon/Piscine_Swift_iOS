Application for creating and manipulating  
shapes. Shapes are created with randon shape  
(square or circle) and color. They can be  
dragged, scaled and rotated.  
  
**Tags:**  
`Dynamic Behavior`, `Gestures Recognizer`, `CoreMotion`
